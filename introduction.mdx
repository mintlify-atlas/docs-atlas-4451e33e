# n8n-nodes-azuracast

`@renebello/n8n-nodes-azuracast` is an n8n community node that connects [AzuraCast](https://www.azuracast.com/) with n8n workflows through the official AzuraCast Web API.

[AzuraCast](https://www.azuracast.com/) is a self-hosted web radio platform for managing stations, media, playlists, requests, webhooks, reports, and admin operations. [n8n](https://n8n.io/) is a workflow automation platform that lets you connect services and automate tasks with a visual editor.

## What is this node?

This package provides a single **AzuraCast** node that exposes every operation from the official AzuraCast OpenAPI specification. Instead of making raw HTTP calls to your AzuraCast server, you can select a resource and action directly in the n8n node editor — with all required and optional fields automatically rendered based on the OpenAPI metadata.

## Coverage

## Key features

- **263/263 OpenAPI operation coverage** — every AzuraCast endpoint is available
- **Unified single node** — operations grouped by resource tag in one n8n node
- **Auto-rendered fields** — path, query, and body parameters from OpenAPI metadata
- **All request body types** — JSON, raw text, binary file upload, multipart form-data
- **Smart response handling** — auto-detects JSON, text, or binary response format
- **Public + protected endpoints** — works with or without an API key
- **Backward compatible** — legacy `operationId` and JSON parameter workflows continue to work
- **AI tool support** — usable as a tool node in n8n AI agent workflows
