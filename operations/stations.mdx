---
title: Stations operations
description: Manage station content, configuration, and broadcasting
---

<Note>
  All Stations operations require a valid API key with station-level or higher permissions. Configure your AzuraCast credentials in n8n before using these operations.
</Note>

Stations operations cover the full lifecycle of managing a radio station: uploading and organising media, building playlists, controlling the broadcast backend, managing live DJ accounts, publishing podcasts, configuring mount points and HLS streams, viewing reports, and more.

<AccordionGroup>
  <Accordion title="Stations General">
    Station profile management, log access, quota information, custom assets, and fallback track configuration.

    | Operation | Method | Path | Description |
    |-----------|--------|------|-------------|
    | Get station profile | `GET` | `/station/{station_id}/profile` | Retrieve the public profile of the station |
    | Get editable profile | `GET` | `/station/{station_id}/profile/edit` | Get the editable station profile |
    | Save station profile | `PUT` | `/station/{station_id}/profile/edit` | Save changes to the station profile |
    | Update now playing metadata | `POST` | `/station/{station_id}/nowplaying/update` | Manually update the Now Playing metadata |
    | Get quota | `GET` | `/station/{station_id}/quota` | Get current usage and quota for the station storage |
    | Get quota by type | `GET` | `/station/{station_id}/quota/{type}` | Get quota for a specific storage location type |
    | List station logs | `GET` | `/station/{station_id}/logs` | Return available log types for the station |
    | View station log | `GET` | `/station/{station_id}/log/{key}` | View the contents of a specific station log |
    | Get custom asset | `GET` | `/station/{station_id}/custom_assets/{type}` | Get the details of a custom asset |
    | Upload custom asset | `POST` | `/station/{station_id}/custom_assets/{type}` | Upload a new custom asset |
    | Delete custom asset | `DELETE` | `/station/{station_id}/custom_assets/{type}` | Remove a custom asset |
    | Get fallback track | `GET` | `/station/{station_id}/fallback` | Get the custom fallback track |
    | Upload fallback track | `POST` | `/station/{station_id}/fallback` | Set a new custom fallback track |
    | Delete fallback track | `DELETE` | `/station/{station_id}/fallback` | Remove the custom fallback track |
  </Accordion>

  <Accordion title="Stations Media">
    Upload, manage, and organise media files in the station library. Supports individual file operations, bulk CSV imports, directory management, waveform data, and batch actions.

    | Operation | Method | Path | Description |
    |-----------|--------|------|-------------|
    | List files | `GET` | `/station/{station_id}/files` | List all uploaded media files |
    | List files by path | `GET` | `/station/{station_id}/files/list` | List files in a media directory by path |
    | Get file | `GET` | `/station/{station_id}/file/{id}` | Retrieve details for a single file |
    | Upload file (multipart) | `POST` | `/station/{station_id}/files` | Upload a new media file |
    | Upload and process file | `POST` | `/station/{station_id}/files/upload` | Upload and process a new media file |
    | Edit file | `PUT` | `/station/{station_id}/file/{id}` | Update metadata for a single file |
    | Delete file | `DELETE` | `/station/{station_id}/file/{id}` | Delete a single file |
    | Play file by ID | `GET` | `/station/{station_id}/file/{id}/play` | Download or stream a file by ID |
    | Play file by song ID | `GET` | `/station/{station_id}/file/{song_id}/play` | Download or stream a file by song unique ID |
    | Download file by path | `GET` | `/station/{station_id}/files/download` | Download a file by relative path |
    | List directories | `GET` | `/station/{station_id}/files/directories` | List directories for moving or renaming |
    | Create directory | `POST` | `/station/{station_id}/files/mkdir` | Create a new directory in the media library |
    | Rename file or directory | `PUT` | `/station/{station_id}/files/rename` | Rename specified files or directories |
    | Batch action | `PUT` | `/station/{station_id}/files/batch` | Perform a batch action on files or directories |
    | Get media art | `DELETE` | `/station/{station_id}/art/{media_id}` | Remove the album art for a track |
    | Set media art | `POST` | `/station/{station_id}/art/{media_id}` | Upload album art for a track |
    | Get waveform data | `GET` | `/station/{station_id}/waveform/{media_id}` | Get cached waveform data for the Visual Cue Editor |
    | Save waveform data | `POST` | `/station/{station_id}/waveform/{media_id}` | Save cached waveform data |
    | Bulk media download | `GET` | `/station/{station_id}/files/bulk` | Download a CSV of all station media |
    | Bulk media preview | `POST` | `/station/{station_id}/files/bulk/preview` | Preview changes from a bulk media CSV |
    | Bulk media upload | `POST` | `/station/{station_id}/files/bulk` | Upload a CSV to update station media in bulk |
  </Accordion>

  <Accordion title="Stations Playlists">

    | Operation | Method | Path | Description |
    |-----------|--------|------|-------------|
    | List playlists | `GET` | `/station/{station_id}/playlists` | List all current playlists |
    | Get playlist | `GET` | `/station/{station_id}/playlist/{id}` | Retrieve details for a single playlist |
    | Create playlist | `POST` | `/station/{station_id}/playlists` | Create a new playlist |
    | Edit playlist | `PUT` | `/station/{station_id}/playlist/{id}` | Update playlist details |
    | Delete playlist | `DELETE` | `/station/{station_id}/playlist/{id}` | Delete a single playlist |
    | Toggle playlist | `PUT` | `/station/{station_id}/playlist/{id}/toggle` | Enable or disable a playlist |
    | Empty playlist | `DELETE` | `/station/{station_id}/playlist/{id}/empty` | Remove all songs from a song-based playlist |
    | Clone playlist | `POST` | `/station/{station_id}/playlist/{id}/clone` | Create a copy of a playlist |
    | Export playlist | `GET` | `/station/{station_id}/playlist/{id}/export/{format}` | Export playlist in PLS or M3U format |
    | Import playlist | `POST` | `/station/{station_id}/playlist/{id}/import` | Import songs from a PLS or M3U file |
    | Get playlist order | `GET` | `/station/{station_id}/playlist/{id}/order` | Get sequential track order |
    | Set playlist order | `PUT` | `/station/{station_id}/playlist/{id}/order` | Set sequential track order |
    | Get playback queue | `GET` | `/station/{station_id}/playlist/{id}/queue` | Get the current playback queue for the playlist |
    | Clear playback queue | `DELETE` | `/station/{station_id}/playlist/{id}/queue` | Reset the internal playback queue |
    | Reshuffle playlist | `PUT` | `/station/{station_id}/playlist/{id}/reshuffle` | Re-shuffle a shuffled playlist |
    | Get apply-to directories | `GET` | `/station/{station_id}/playlist/{id}/apply-to` | Get directories the playlist can be applied to |
    | Apply playlist to directories | `PUT` | `/station/{station_id}/playlist/{id}/apply-to` | Apply the playlist to specified directories |
  </Accordion>

  <Accordion title="Stations Broadcasting">

    | Operation | Method | Path | Description |
    |-----------|--------|------|-------------|
    | Get service status | `GET` | `/station/{station_id}/status` | Retrieve the current status of all broadcast services |

    | Control backend (Liquidsoap) | `POST` | `/station/{station_id}/backend/{action}` | Perform service actions on the radio backend |
    | Control frontend | `POST` | `/station/{station_id}/frontend/{action}` | Perform service actions on the radio frontend |
    | Get Liquidsoap config | `GET` | `/station/{station_id}/liquidsoap-config` | Get the generated and editable Liquidsoap config |
    | Save Liquidsoap config | `PUT` | `/station/{station_id}/liquidsoap-config` | Save the editable sections of the Liquidsoap config |
    | Export Liquidsoap config | `GET` | `/station/{station_id}/liquidsoap-config/export` | Download an archive of all Liquidsoap configuration |
    | Import Liquidsoap config | `POST` | `/station/{station_id}/liquidsoap-config/import` | Restore a previously exported Liquidsoap config archive |
    | Get Stereo Tool config | `GET` | `/station/{station_id}/stereo-tool-configuration` | Get the Stereo Tool configuration file |
    | Upload Stereo Tool config | `POST` | `/station/{station_id}/stereo-tool-configuration` | Update the Stereo Tool configuration file |
